## 🔧 配置步骤

### 第一步：配置Unsplash API

#### 1. 注册Unsplash开发者账号

访问：https://unsplash.com/developers

1. 点击 "Register as a developer"
2. 创建一个新应用（New Application）
3. 填写应用信息：
   - Application name: `Agent Article Publisher`
   - Description: `Automatic cover image for articles`
4. 同意条款，创建应用

#### 2. 获取Access Key

创建应用后，你会看到：
- **Access Key** - 这是你需要的
- Secret Key - 不需要

复制 Access Key（类似：`abc123xyz...`）

#### 3. 配置环境变量

编辑 `/Users/yvonne/Documents/Agent/.env` 文件，添加：

```bash
UNSPLASH_ACCESS_KEY=你的Access_Key
```

**示例：**
```bash
UNSPLASH_ACCESS_KEY=abc123xyz456def789ghi
```

#### 4. 测试配置

运行Agent，如果配置正确，你会看到：
```
🔍 正在 Unsplash 搜索关键词: AI革命来临
📸 找到图片，摄影师: John Doe
✅ 封面图已保存: output/final_published/xxx_cover.jpg
```

如果未配置，会显示：
```
⚠️ 未配置 UNSPLASH_ACCESS_KEY，跳过封面图下载
```

### 第二步：配置微信公众号自动发布

#### 1. 安装依赖

##### macOS:
```bash
# 安装 Node.js（如果还没安装）
brew install node

# 安装 Bun（JavaScript运行时）
curl -fsSL https://bun.sh/install | bash

# 安装 Chrome 浏览器（如果还没安装）
# 从 https://www.google.com/chrome/ 下载安装
```

##### Linux:
```bash
# 安装 Node.js
sudo apt-get install nodejs npm

# 安装 Bun
curl -fsSL https://bun.sh/install | bash

# 安装 Chrome
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb

# 安装 xdotool（用于键盘模拟）
sudo apt-get install xdotool
```

#### 2. 验证安装

```bash
# 检查 Node.js
node --version  # 应该显示版本号，如 v18.x.x

# 检查 Bun
bun --version  # 应该显示���本号

# 检查 Chrome
google-chrome --version  # 应该显示版本号
```

#### 3. 配置 baoyu-skills

baoyu-skills 已经下载到：
```
/Users/yvonne/Documents/Agent/baoyu-skills/
```

不需要额外配置，首次运行时会自动安装依赖。

#### 4. 首次使用流程

当你运行Agent并执行到Step 7时：

1. **浏览器会自动打开**
   - 打开微信公众号后台：https://mp.weixin.qq.com/

2. **扫码登录**
   - 用微信扫描二维码登录
   - ⚠️ 重要：登录后不要关闭浏览器

3. **自动发布**
   - Agent会自动：
     - 点击"文章"菜单
     - 粘贴文章内容
     - 上传图片
     - 填写标题、作者、摘要
     - 保存为草稿

4. **完成**
   - 文章会保存为草稿
   - 你可以在微信公众号后台预览和发布

## 🐛 常见问题

### 问题1：Unsplash API 请求失败

**原因：** Access Key 配置错误或API额度用完

**解决：**
1. 检查 `.env` 文件中的 `UNSPLASH_ACCESS_KEY` 是否正确
2. Unsplash 免费账号每小时限制50次请求
3. 如果超限，等待1小时后重试

### 问题2：未找到封面图

**原因：** 搜索关键词太具体，没有匹配的图片

**解决：**
- Agent会自动跳过，不影响后续流程
- 可以手动添加封面图到生成的文档中

### 问题3：微信发布失败 - 未安装 Node.js

**错误信息：** `❌ 未安装 Node.js 或 Bun，无法执行发布`

**解决：**
```bash
# macOS
brew install node

# Linux
sudo apt-get install nodejs npm
```

### 问题4：微信发布超时

**原因：** 需要手动扫码登录，但超过5分钟未操作

**解决：**
1. 确保在5分钟内完成扫码
2. 如果超时，文章已保存为Word文档，可以手动复制粘贴

### 问题5：Chrome 浏览器未找到

**错误信息：** 浏览器启动失败

**解决：**
```bash
# 设置 Chrome 路径（如果安装在非标准位置）
export WECHAT_BROWSER_CHROME_PATH="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"
```

## 📈 性能优化

### Unsplash 图片质量

默认下载 `regular` 尺寸（1080px宽），如需更高质量：

编辑 `src/tools.py`:
```python
image_url = photo["urls"]["full"]  # 原图（大文件）
# 或
image_url = photo["urls"]["regular"]  # 1080px（推荐）
# 或
image_url = photo["urls"]["small"]  # 400px（快速）
```

### 微信发布速度

如果发布速度慢，可以：
1. 使用更快的网络
2. 减少图片数量
3. 压缩图片大小
