# 微信公众号发布工具使用说明

## 📱 功能说明

这是一个独立的微信公众号文章发布脚本，可以将 Markdown 文件自动发布到微信公众号。

## 🚀 使用方法

### 方法1：交互式模式（推荐）

直接运行脚本，会列出所有已生成的文章供你选择：

```bash
cd /Users/yvonne/Documents/Agent
python3 publish_to_wechat.py
```

然后按照提示：
1. 选择要发布的文章编号
2. 选择主题样式（可选）
3. 确认发布

### 方法2：命令行模式

直接指定文件路径：

```bash
# 使用默认主题 (grace)
python3 publish_to_wechat.py output/final_published/PUBLISH_20260125_183148_ID0_被访者为数学家兼哲学.md

# 指定主题
python3 publish_to_wechat.py output/final_published/PUBLISH_20260125_183148_ID0_被访者为数学家兼哲学.md cyan
```

### 方法3：使用相对路径

```bash
python3 publish_to_wechat.py "output/final_published/PUBLISH_*.md"
```

## 🎨 可用主题

- `grace` - 优雅风格（默认）
- `cyan` - 青色
- `purple` - 紫色
- `green` - 绿色
- `orange` - 橙色
- `red` - 红色
- `blue` - 蓝色

## ⚠️ 注意事项

1. **首次使用需要扫码登录**
   - 脚本��自动打开浏览器
   - 使用微信扫描二维码登录公众号后台

2. **发布过程中请勿操作浏览器**
   - 等待自动化流程完成
   - 通常需要 1-2 分钟

3. **确保已安装依赖**
   - Node.js 或 Bun
   - Chrome 浏览器
   - baoyu-skills 工具包

## 🔧 前置要求

### 安装 Node.js
```bash
# macOS
brew install node

# 或访问 https://nodejs.org/
```

### 安装 Bun（可选，更快）
```bash
curl -fsSL https://bun.sh/install | bash
```

### 检查 baoyu-skills
确保以下路径存在：
```
Agent/baoyu-skills/skills/baoyu-post-to-wechat/scripts/wechat-article.ts
```

## 🐛 常见问题

### 1. 提示"未找到 baoyu-post-to-wechat 脚本"
**解决方案**：检查 `baoyu-skills` 文件夹是否存在且包含正确的脚本文件

### 2. 提示"未安装 Node.js 或 Bun"
**解决方案**：安装 Node.js 或 Bun

### 3. 发布超时
**解决方案**：
- 检查网络连接
- 手动在浏览器中扫码登录
- 增加超时时间（修改脚本中的 timeout 参数）

### 4. 浏览器无法启动
**解决方案**：
- 确保 Chrome 浏览器已安装
- 检查浏览器版本是否过旧
- 尝试重启电脑
